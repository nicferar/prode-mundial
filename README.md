# PRODEAR

El juego de predicciones para tu grupo.

Pronosticá resultados del Mundial, competí con amigos y trepá en la tabla.

## Stack

React · Vite · Firebase · react-router-dom

## Arrancar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Firebase

```bash
npm install -g firebase-tools
firebase login
firebase deploy
```
